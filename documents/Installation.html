Installation

Installing Virtual Chat is a simple process. Please visit the install folder to begin. This is located at `http://yoursite.com/install`, assuming you have installed our product in the root directory. Simply append /install/ to the end of the URL.

Please ensure `core/rc` is writable and that you have a database ready to use. For installation support, please contact DesignSkate via CodeCanyon or www.DesignSkate.com.

Server Requirements

  • PHP 5.3 & beyond
  • One MySQL database
  • MySQLi extension for PHP enabled
