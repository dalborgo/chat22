  • 1) Room Coords
    • Room Coordinates

    To create room coordinates, you can use any image map polygon generator, we'll link to a few in this guide. After obtaining the coordinates, which will look something like this "101,382,171,928,11", you can enter this into the room file which is located in "static/rooms/lobby/config.xml".

    By default, the line which you must edit looks like this.

    static/rooms/lobby/room_bg.png|123, 273, 219, 305, 222, 362, 296, 380, 284, 452, 308, 470, 350, 435, 425, 483, 382, 524, 407, 535, 465, 493, 508, 516, 542, 489, 597, 486, 614, 445, 651, 405, 698, 375, 770, 406, 790, 383, 800, 351, 842, 339, 754, 330, 714, 232, 621, 185, 549, 216, 397, 141, 325, 173, 392, 209, 295, 255

    To create the new coordinates, simply replace the existing ones. The new line should look like this (coordinates are just an example).

    static/rooms/lobby/room_bg.png|101,382,171,928,11

    You may have noticed the | character, this is what we use to seperate content and parse the XML within JavaScript. Without this seperator, the chat room will not function correctly.
